Smart Expense Manager

A complete Flutter expense tracking app built with Hive for offline database, charts, categories, and PDF export. Integrated with FastAPI backend for optional cloud sync and authentication. Built with a clean, modern Flutter UI and dark/light themes.

Key Features

  • Offline-first storage using Hive / SQLite
  • Interactive expense charts and summaries
  • Category-based and daily expense grouping
  • PDF export and share functionality
  • Dark / Light theme with dynamic UI
  • FastAPI backend integration (sync & auth)
